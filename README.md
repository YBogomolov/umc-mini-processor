# UMC Mini Processor

## Setup

1. `uv sync`.
2. Open terminal, activate venv: `source .venv/bin/activate`.

## Usage

1. Download a collection ZIP from [UMC](https://ybogomolov.me/umc), extract it.
2. `python processor.py <path to extracted folder>`.

Or:

1. Build the app: `make build`
2. Open `dist`, run the app.
3. Drag'n'drop the root folder of a mini collection. The resulting PDF will be open automatically.
